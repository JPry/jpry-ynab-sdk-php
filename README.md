# JPry YNAB

Framework-agnostic PHP library for YNAB.

## Goals

- Keep integration independent from Tempest or any framework.
- Provide typed models for budgets, transactions, accounts, categories, and payees.
- Support API key auth and OAuth tokens.
- Use Guzzle by default when available, while allowing any custom HTTP sender implementation.

## Status

This package is in active development and not yet published.
